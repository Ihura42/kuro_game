# 2D Platformer hra v Pygame

Tato aplikace je jednoduchá 2D platformer hra vytvořená v Pythonu s využitím knihovny Pygame. Hráč ovládá postavu jménem Kuro, která se pohybuje po herním světě, skáče a útočí na nepřátele.

## Jak hrát

1. **Ovládání postavy:**
   - **A/D klávesy:** Pohyb vlevo/vpravo.
   - **W/Mezerník klávesa:** Skok.
   - **K klávesa:** Útok.
   
2. **Cíl hry:**
   - Prozkoumejte herní svět, skákejte přes překážky

## Spuštění

1. Po instalaci knihoven spusťte soubor `main.py`.
2. Hra by se měla spustit a zobrazit se v novém okně.
